What is the difference between Margin, Border, and Padding?

A technical challenge from my second sprint

What do these terms mean?

Margin, border and padding are all terms used to describe and define the space around content and they all surround the content.

If you start in the centre with the content, the first layer surrounding the content is the padding. The next layer is the border and the final outer layer is the margin. In image 1 you can see the "content" surrounded by the padding, border and margin.

When would we use them?

Padding is the first layer that might be wrapped around some content. Padding is space between the content and the border. Padding is used to put room between the content and the box of the content's defined dimensions.

The order is wrapped around the padding. If there is no padding then the content will be right up against the border. Borders can be coloured and dashed and have many other effects added. Borders are used to give an image emphasis or style.

The margin is wrapped around the border. The border and the padding are both within the defined dimensions of the content. The margin is outside these defined dimensions. You use margins to give the content room between other items of content. You can think of margins as a bit like personal space, we don't want people entering our personal space!

How can we use them?

Padding, border and margin can all be defined and styled using CSS. You can change the value for each layer in a uniform way, or you can change the value for each side individually. Images 2-4 show a few different examples of using padding, border and margin.

The light pink lines show the borders of the content and the borders of the div that the content is inside of. Image 2 has space between the image and the border (padding), a pretty dashed purple border, and space between the outside of the content and the div it is nesting inside. You could change the colour of the padding by giving the image a background colour. Image 3 has no space between the border and the image because it has no padding. Image 4 has no border and no margin, but there is still a gap between the image and the box defining the dimensions of the image because it has padding. Image 4 is touching the top of the div it is nested in because the image has no margin.
